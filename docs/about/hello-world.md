# 第一个 Markdown 文档

## Hello.md

打开你任何文本编辑器（我习惯使用的是 [VS Code](https://code.visualstudio.com/)），新建一个 'hello.md' 文件，写入下面的内容，并保存：

```markdown
# Hello World

Learning coding needs [Google](https://www.google.com/)

Coding needs:
- a computer
- a keyboard
    - ctrl + c
    - ctrl + v
```

> '.md' 和 '.markdown' 都是被普遍支持的扩展名，不过 '.md' 更加简单和方便。

这段文字将会被转换成下面的样子👇

# Hello World

Learning coding needs [Google](https://www.google.com/)

Coding needs:
- a computer
- a keyboard
    - ctrl + c
    - ctrl + v
